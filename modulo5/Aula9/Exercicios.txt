a)

```sql
ALTER TABLE Actor DROP COLUMN salary;
```
	Este comando altera a tabela Actor apagando o campo salary.

b)

```sql
ALTER TABLE Actor CHANGE gender sex VARCHAR(6);
```

	Este comando altera a tabela Actor alterando o campo gender por sex VARCHAR(6)

c)

```sql
ALTER TABLE Actor CHANGE gender gender VARCHAR(255);
```

	Este comando altera a tabela Actor trocando a "estrutura" do campo gender para o tipo VARCHAR(255) (tamanho 255).
